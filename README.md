# 📊 SCT_DA_1 - Excel Sales Dashboard

## 📌 Project Overview

This project was completed as part of the SkillCraft Technology Data Analytics Internship. The objective was to analyze sales data and create an interactive dashboard using Microsoft Excel.

## 🎯 Objectives

* Analyze total sales and profit performance
* Visualize sales trends over time
* Compare category-wise sales and profits
* Identify high and low performing segments

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting

## 📈 Dashboard Insights

* Total Sales Analysis
* Profit by Category
* Sales Trend Analysis
* Performance Monitoring
* Interactive Visualizations
